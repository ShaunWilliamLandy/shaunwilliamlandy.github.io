LandHawk

LandHawk is a web application that allows users to find and assess land development opportunities. It combines hundreds of disparate data sources to provide powerful geospatial insights for millions of sites across the UK.

Read more about LandHawk here

Some key features that I worked on:

  • Development of a search feature that would allow users to search large areas and refine their results using a custom query builder.
  • Creation of an info panel that compiles together large amounts of geospatial information for a parcel of land into a simple to understand interface.
  • Construction of a kanban board to allow a user to organise their sites and a dashboard to view their site information.
  • Built printout functionality that allowed users to print out maps or reports as pdfs.
  • Refactored existing frontend after a UI rebrand using designs from Figma.
